# 高级进阶

本页面深入探索高级特性与最佳实践。

## 插件系统

Docsify 支持丰富的插件生态：

| 插件 | 功能 |
|------|------|
| search | 全文搜索 |
| emoji | Emoji 支持 |
| external-link | 外链检测 |
| copy-code | 代码复制按钮 |

## 自定义主题

切换主题只需更改 CSS 引用：

```html
<!-- 默认 Vue 主题 -->
<link rel="stylesheet" href="//cdn.jsdelivr.net/npm/docsify@4/lib/themes/vue.css">

<!-- 黑色主题 -->
<link rel="stylesheet" href="//cdn.jsdelivr.net/npm/docsify@4/lib/themes/dark.css">
```

## 部署到公网

Docsify 作为纯静态站点，可以部署到任何静态托管平台。

> 本站点正是通过 CloudStudio 部署到公网的！
