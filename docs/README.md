# 欢迎使用 Docsify 文档站点

## 简介

这是一个使用 **Docsify** 构建的文档站点示例，包含三个相互链接的 Markdown 文件。

Docsify 是一个神奇的文档站点生成器，它不会将 Markdown 文件编译成 HTML，而是直接在浏览器中动态渲染 Markdown 内容。

## 核心特性

- **实时渲染** - 无需构建，修改即生效
- **轻量级** - 只需要一个 HTML 入口文件
- **支持 Markdown** - 直接编写 .md 文件
- **内置搜索** - 自动生成全文搜索
- **主题丰富** - 支持多种主题切换
